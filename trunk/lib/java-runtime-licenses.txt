This file contains copyright and license declarations
of libraries shipped in the java/runtime/ subdirectory.
It's main purpose is that the reuse tool picks up all licenses
and adds them to the LICENSE folder,
which helps when creating a CPAchecker release archive.
(For this, the file size needs to be less than 4kB,
so we split it.)
